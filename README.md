# Three.js Music Visualizer

A full-screen, interactive, audio-reactive visualizer built with Three.js and Vite.

## Features
- **Real-time Audio Analysis**: Sensitive beat detection and frequency analysis.
- **Multiple Visual Presets**:
  - **Grid Tunnel**: Synthwave-style terrain.
  - **Pentagonal Core**: Abstract geometry.
  - **Circular Wave**: Pulse rings.
- **Interactive UI**: Track selection, play controls, and stats.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open the link provided (usually `http://localhost:5173`).

## Usage
- Click anywhere to initialize the audio context.
- Use the UI to switch tracks.
- The visualizer automatically switches styles based on the track genre.
